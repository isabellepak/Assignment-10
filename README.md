# Module 10: Data Manipulation and Cleaning – UrbanStyle Customer Data Cleaning

## Files Included
module10-assignment.py

## What I Practiced
In this assignment, I practiced cleaning and preparing raw data using Python. I used pandas and numpy to handle missing values, correct data types, standardize text, and remove duplicates. I also developed skills in transforming messy datasets into clean, structured data ready for analysis and generating basic business insights.

## Key Concepts
Data cleaning and preprocessing  
Handling missing values and duplicates  
Data type conversion (dates and numeric values)  
Text standardization and formatting  
Feature engineering and derived metrics  
Basic data analysis using pandas  

## Summary
This project demonstrates how raw data can be cleaned and transformed into a usable format for analysis. The final dataset provides clearer insights into customer behavior, spending patterns, and loyalty trends, supporting more accurate and reliable business decision-making.
